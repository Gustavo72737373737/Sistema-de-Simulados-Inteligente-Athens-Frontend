📘 Documentação do JSON - Athens

Esta página descreve apenas a estrutura do JSON utilizada pelo sistema. Os valores mostrados são apenas marcadores (placeholders) e devem ser substituídos pelos dados reais.

Estrutura Geral

{
  "titulo": "{titulo}",
  "materia": "{materia}",
  "tema": "{tema}",

  "questoes": [

  ]

}

Tipos de Questões

Tipo Descrição
multipla_escolha Questão com alternativas.
verdadeiro_falso Questão com duas opções: Verdadeiro ou Falso.
resposta_curta Campo de texto de uma linha.
discursiva Campo de texto longo.

Estrutura de cada tipo

1. Múltipla escolha

{
  "id": 1,
  "tipo": "multipla_escolha",
  "pergunta": "{pergunta}",

  "alternativas": [
    "{alternativa_1}",
    "{alternativa_2}",
    "{alternativa_3}",
    "{alternativa_4}"
  ]
}

2. Verdadeiro ou Falso

{
  "id": 2,
  "tipo": "verdadeiro_falso",
  "pergunta": "{pergunta}"
}

3. Resposta Curta

{
  "id": 3,
  "tipo": "resposta_curta",
  "pergunta": "{pergunta}"
}

4. Discursiva

{
  "id": 4,
  "tipo": "discursiva",
  "pergunta": "{pergunta}"
}

Estrutura Completa

{
  "titulo": "{titulo}",

  "materia": "{materia}",

  "tema": "{tema}",

  "questoes": [

    {
      "id": 1,
      "tipo": "multipla_escolha",
      "pergunta": "{pergunta}",

      "alternativas": [
        "{alternativa_1}",
        "{alternativa_2}",
        "{alternativa_3}",
        "{alternativa_4}"
      ]
    },

    {
      "id": 2,
      "tipo": "verdadeiro_falso",
      "pergunta": "{pergunta}"
    },

    {
      "id": 3,
      "tipo": "resposta_curta",
      "pergunta": "{pergunta}"
    },

    {
      "id": 4,
      "tipo": "discursiva",
      "pergunta": "{pergunta}"
    }

  ]
}

Campos

Campo Obrigatório Descrição
titulo Sim Nome do simulado.
materia Sim Nome da matéria.
tema Sim Tema do simulado.
questoes Sim Lista contendo todas as questões.
id Sim Identificador único da questão.
tipo Sim Tipo da questão.
pergunta Sim Texto da pergunta.
alternativas Apenas múltipla escolha Lista de alternativas.

Regras

  • Cada questão deve possuir um id único.
  • O campo tipo deve conter exatamente um dos quatro tipos aceitos.
  • A propriedade alternativas existe apenas para questões de múltipla escolha.
  • Utilize sempre aspas duplas (").
  • Não esqueça vírgulas entre os elementos.
  • Os textos entre chaves, como {pergunta}, são apenas marcadores e devem ser substituídos.

JSON Inválido

{
  "alternativas": [
    "{alternativa_1}"
    "{alternativa_2}"
  ]
}

❌ Erro: falta uma vírgula entre os elementos do array.

Fluxo de Uso

  1. Monte o JSON seguindo esta documentação.
  2. Abra o Athens.
  3. Cole o JSON no campo de entrada.
  4. Clique em Carregar Simulado.
  5. Responda às questões.
  6. Exporte o JSON contendo apenas as respostas.

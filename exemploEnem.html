📘 Documentação do JSON - Modo ENEM (Athens)

Esta página descreve a estrutura de dados obrigatória para rodar simulados no Modo ENEM. Garante a renderização em formato de folha de papel limpa do Word, itálicos automáticos para referências e folheamento por páginas.

Estrutura Geral do Modo ENEM

Diferente do simulado padrão, o formato ENEM recebe uma lista direta (Array) de objetos de questões no arquivo, sem chaves de topo de metadados externos.

[
  {
    "id": 1,
    "titulo": "{titulo_da_questao}",
    "disciplina": "{disciplina}",
    "contextoParagrafos": [
      "{paragrafo_1}",
      "{paragrafo_2}"
    ],
    "enunciado": "{enunciado_comando}",
    "gabarito": "{letra_correta}",
    "imagemUrl": "{link_da_imagem_ou_vazio}",
    "alternativas": [
      { "letra": "A", "texto": "{texto_A}" },
      { "letra": "B", "texto": "{texto_B}" }
    ]
  }
]

Formato dos Atributos Específicos

Veja como o interpretador do ENEM monta cada pedaço da folha de impressão baseando-se nos atributos:

1. Divisão de Parágrafos e Bibliografia

O campo contextoParagrafos aceita uma lista de textos corridos. O sistema lê as linhas e, se identificar termos como "Disponível em" ou "Adaptado", aplica automaticamente as regras de itálico menor daABNT.

"contextoParagrafos": [
  "Sentimos que toda satisfação de nossos desejos advinda do mundo assemelha-se à esmola...",
  "SCHOPENHAUER, A. Aforismo para a sabedoria da vida. São Paulo: Martins Fontes, 2005."
]

2. Controle de Imagens

Se a questão possuir figuras ou gráficos, passe a URL direta em imagemUrl. Caso não possua, defina como uma string vazia ("") para o carregador ocultar a tag da folha perfeitamente.

"imagemUrl": "https://enem.dev"

3. Array de Objetos nas Alternativas

No modo ENEM, cada alternativa é um objeto contendo obrigatoriamente as chaves letra e texto:

"alternativas": [
  { "letra": "A", "texto": "Consagração de relacionamentos afetivos." },
  { "letra": "B", "texto": "Administração da independência interior." }
]

Modelo de Estrutura Completa (Exemplo Aceito)

Copie o modelo abaixo para usar como base na criação de novas questões estruturadas:

[
  {
    "id": 1,
    "titulo": "Questão 1 - ENEM 2016",
    "disciplina": "ciencias-humanas",
    "contextoParagrafos": [
      "Sentimos que toda satisfação de nossos desejos advinda do mundo assemelha-se à esmola que mantém hoje o mendigo vivo, porém prolonga amanhã a sua fome.",
      "SCHOPENHAUER, A. Aforismo para a sabedoria da vida. São Paulo: Martins Fontes, 2005."
    ],
    "enunciado": "O trecho destaca uma ideia remanescente de uma tradição filosófica ocidental, segundo a qual a felicidade se mostra indissociavelmente ligada à",
    "gabarito": "B",
    "imagemUrl": "",
    "alternativas": [
      { "letra": "A", "texto": "Consagração de relacionamentos afetivos." },
      { "letra": "B", "texto": "Administração da independência interior." },
      { "letra": "C", "texto": "Fugacidade do conhecimento empírico." },
      { "letra": "D", "texto": "Liberdade de expressão religiosa." },
      { "letra": "E", "texto": "Busca de prazeres efêmeros." }
    ]
  }
]

Mapeamento de Campos (Tabela de Parâmetros)

Campo Obrigatório Tipo de Dado Descrição
id Sim Número Posição ordinal da questão.
titulo Sim Texto Título de cabeçalho da questão (Ex: Questão 1 - ENEM 2016).
disciplina Sim Texto Matéria ou área do conhecimento (Ex: ciencias-humanas).
contextoParagrafos Sim Array de Textos Lista contendo os blocos de texto de apoio da questão.
enunciado Sim Texto O comando/pergunta final da questão.
gabarito Sim Texto Letra da alternativa correta (Utilizado para o cálculo de acertos).
imagemUrl Sim Texto (URL) Link completo da imagem. Envie string vazia "" se não houver.
alternativas Sim Array de Objetos Lista contendo as chaves ordinais "letra" e "texto".

Regras Fundamentais do Modo ENEM

  • O arquivo DEVE começar com colchetes [ e terminar com colchetes ].
  • O campo contextoParagrafos não aceita strings puras, precisa ser uma lista de strings separadas por vírgula.
  • O identificador de gabarito deve ser em letra maiúscula ("A", "B", etc).
  • Não use sintaxe Markdown para injetar imagens no meio do texto; use sempre o campo isolado imagemUrl.

Exemplo de Erro Comum (JSON Inválido)

{
  "titulo": "Questão 1",
  "contextoParagrafos": "Texto corrido direto sem ser lista"
}

❌ Erro: O carregador do ENEM utiliza repetições (forEach) para renderizar os parágrafos. Se o dado não for enviado dentro de um array [ ], o sistema falhará ao carregar.
